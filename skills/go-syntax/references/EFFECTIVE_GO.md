# Effective Go Patterns

## Naming Conventions
- Package names: lowercase, single word
- Exported: PascalCase
- Unexported: camelCase
- Interfaces: -er suffix (Reader, Writer)

## Code Organization
- Keep functions small
- Return early (guard clauses)
- Group related declarations
