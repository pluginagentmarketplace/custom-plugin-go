# Go Toolchain

## Essential Tools
- `go build` - Compile
- `go run` - Compile and run
- `go test` - Testing
- `go fmt` - Format code
- `go vet` - Static analysis
- `go mod` - Module management
- `go doc` - Documentation

## Third-Party
- golangci-lint - Linting
- delve - Debugging
- air - Hot reload
