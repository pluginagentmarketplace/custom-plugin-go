# Learning Workflows

## User Journey Maps

### New Learner (0 experience)

```
Day 1:
  /explore → Browse all domains
  → Assess: Self-assessment via /assess
  → Choose interest (web, data, AI, etc)

Week 1-2:
  /learn [domain] → Start learning path
  Agent → Explains fundamentals
  Skill → Code examples and tools
  → Build first tiny project

Week 3-4:
  /projects → Find beginner project
  Build → Code along with tutorial
  Test → Run local tests
  → Iterate and improve

Month 2:
  Build 2-3 more projects
  Read agent deep-dives
  Follow recommended learning timeline
  Join community

Month 3:
  /career-path → Plan progression
  Choose specialization
  Build portfolio-worthy project
  Start job search / freelancing
```

### Experienced Developer (wants new skill)

```
Day 1:
  /assess → Self-evaluate
  Identifies gaps
  → Find fastest learning path

Week 1:
  /learn [domain] → Jump to intermediate
  Skip fundamentals
  Focus on frameworks/patterns
  → Build project in new domain

Month 1:
  3-4 practical projects
  Review advanced agent sections
  /projects → Specialized projects
  → Ready for professional use
```

### Career-Focused (wants advancement)

```
Month 1:
  /career-path → Choose path (Manager/Specialist)
  Identify target role
  → Plan 12-36 month progression

Months 1-3:
  /learn [complementary skill]
  Build technical expertise
  Take on leadership opportunities
  → Document results

Months 4-6:
  Lead projects
  Mentor junior developers
  Speak internally
  /projects → Advanced architecture projects

Months 7-12:
  Interview prep: system design
  Network with peers
  Personal brand building
  → Apply for target roles
```

## Recommended Workflows

### Web Development Fast Track (3 months)
```
Week 1-2:   HTML/CSS/JavaScript (Agent 01 Foundation)
Week 3-6:   Choose framework (React/Vue/Angular)
Week 7-10:  Backend integration
Week 11-12: Full-stack project
```

### Full-Stack to System Design (12 months)
```
Month 1-3:  Deepen backend knowledge (Agent 02)
Month 4-6:  Add DevOps/Cloud (Agent 03)
Month 7-9:  Database design (Agent 05)
Month 10-12: System design interview prep
```

### Data Science Career Path (18 months)
```
Month 1-2:  SQL & Databases (Agent 05)
Month 3-4:  Python & Statistics (Agent 02)
Month 5-8:  ML Fundamentals (Agent 04)
Month 9-12: Big Data & Pipelines (Agent 05)
Month 13-18: Specialized ML project
```

## Command Integration

### For Learning
- `/explore` → Discovery phase
- `/learn [domain]` → Structured path
- `/assess` → Self-evaluation
- `/projects` → Hands-on practice

### For Career
- `/career-path` → Progression planning
- `/assess` → Regular skill checks
- `/projects` → Build portfolio
- Agent deep-dives → Stay current

## Best Practices

1. **Start with Assessment**: Know your level
2. **Follow Timeline**: Don't rush fundamentals
3. **Build Projects**: Learn by doing
4. **Join Community**: Network and support
5. **Review Regularly**: Stay on track
6. **Share Knowledge**: Teach others
